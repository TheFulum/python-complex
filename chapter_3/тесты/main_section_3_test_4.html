4. Какие из следующих утверждений о декораторах в Python верны?

   а)Декоратор @classmethod преобразует метод в метод класса, который принимает класс в качестве первого аргумента.

  б) Декоратор @staticmethod преобразует метод в метод экземпляра, автоматически передавая экземпляр класса в качестве первого аргумента.

  в)Декоратор @property используется для создания методов, которые могут быть доступны как атрибуты, без вызова метода.

   г) Декоратор @classmethod используется для создания методов, которые могут быть доступны как атрибуты, без вызова метода.
