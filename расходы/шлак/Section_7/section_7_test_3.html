Тест №3

1. Выберите правильный вариант ответа
Что такое модули ввода-вывода?

  1) Устройства для подключения мониторов и клавиатур.

  2) Устройства для взаимодействия с управляющими системами.

  3) Модули для зарядки мобильных устройств.

  4) Устройства для подключения камер видеонаблюдения.


2. Выберите правильный вариант ответа
Чем отличаются дискретные и аналоговые модули?

  1) Дискретные модули принимают аналоговые сигналы, а аналоговые — цифровые.

  2) Дискретные модули принимают и преобразовывают команды, передаваемые по выбранному протоколу связи.

  3) Аналоговые модули используются только для управления двигателями.

  4) Дискретные модули используются только для измерения температуры.


3. Выберите правильный вариант ответа
Что такое “модульность” в контексте модулей ввода-вы вывода?

  1) Возможность расширения системы дополнительными модулями

  2) Возможность объединения нескольких модулей в одну систему

  3) Возможность изменения конфигурации системы без замены оборудования

  4) Возможность замены отдельных модулей на более современные версии


4. Выберите правильный вариант ответа
Какие характеристики модулей влияют на их производительность?

  1) Только архитектура.

  2) Только количество ядер.

  3) Количество ядер, тактовая частота, архитектура и размер кэш-памяти.

  4) Только размер кэш-памяти.


5. Выберите правильный вариант ответа
Какие особенности имеют модули ввода-вывода?

  1)Высокая точность и стабильность

  2) Возможность работы с различными типами сигналов

  3) Простота использования и настройки

  4) Возможность удаленного управления и мониторинга
